<!Doctype html> Todos los documentos de html empiezan por esta etiqueta, su significado es: el tipo de documento es html.
<html lang=es> Abrimos la etiqueta html para empezar con el documento y le indicamos que el documento estara en español
<head> Abrimos la etiqueta head, en esta etiqueta meteremos metadatos(datos que no se veran en el documento)

<title>Titulo del documento.</title> Para introducir el titulo de la pestaña del navegador.
<meta charset "utf-8"> Para indicar que utilizamos el formato de transformacion unicode de 8 bits.
<meta name="author" content="Nombre Apellido"> Para poner el nombre del creador del documento.

</head> Cerramos la etiqueta head cuando ya no queremos introducir ninguna etiqueta dentro mas

<body> Iniciamos la etiqueta del body, que es donde meteremos todos nuestros datos que queramos que se vean en la pagina web.

<h1>Tipo de titulo</h1>

Tambien podemos utilizar h2 h3 h4 h5 h6 para diferentes tipos de titulos

<p>etiqueta de parrafos.
